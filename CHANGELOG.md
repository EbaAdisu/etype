## [Unreleased]

### Added
- Four game modes: Word Rush, Sentence, Code Snippets, Survival
- XP and level system with difficulty unlocks
- Daily streak tracking with up to ×2.0 XP bonus
- Personal bests tracked per mode × difficulty (16 slots)
- Stats screen with session history and PB grid
- QWERTY key heatmap colored by error rate and keystroke delay
- SQLite persistence at `~/.local/share/etype/etype.db`
- Word lists for Easy / Medium / Hard / Insane difficulties
- Code snippets for Python, Bash, and Rust
- Help screen with keybind reference
